# 测试仪表盘 — 使用说明

## 快速启动

```bash
# 1. 安装依赖（如尚未安装）
pip install flask pytest

# 2. 启动仪表盘
python test/dashboard/test_dashboard.py

# 3. 浏览器打开
#    http://localhost:5000
```

## 功能说明

### 测试案例展示
- 页面加载时展示全部 24 条测试用例（TC-01 ~ TC-24）
- 按 7 个大模块分组展示，每个模块可折叠
- 每条用例显示：编号、中文功能描述、测试状态

### 一键运行
- **运行全部测试**：点击顶部按钮，运行所有 24 条用例
- **运行单个模块**：点击模块右侧的 ▶ 按钮，仅运行该模块下的用例

### 结果查看
- 汇总面板显示：总测试数、通过数、失败数、错误/跳过数、通过率进度条
- 失败/错误用例可点击展开查看错误堆栈

## 文件说明

| 文件 | 用途 |
|------|------|
| `test_dashboard.py` | Flask 应用，提供 Web 界面和 API |
| `test_runner.py` | 测试运行器，调用 pytest 并解析结果 |

## API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | 仪表盘页面 |
| GET | `/api/results` | 获取最近一次测试结果 |
| GET | `/api/status` | 获取运行状态 |
| POST | `/api/run` | 触发测试运行，可选参数 `{"module_class": "TestXxx"}` |
